# AI Sentiment Analysis Project 🚀

This project uses Deep Learning to analyze customer sentiments from text reviews. Developed as part of my deep learning studies.

## 🧠 How it works:
- Uses Hugging Face Transformers pipeline.
- Processes text input to automatically classify sentiment as Positive or Negative.
- Built, tested, and deployed entirely using Cloud & Mobile environments.

## 🛠️ Technologies Used:
- Python
- Google Colab (Cloud UI)
- Hugging Face Transformers (NLP)
